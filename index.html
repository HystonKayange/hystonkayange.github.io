Mthandizi: Communication Tool for the Deaf

Mthandizi translates hand gestures into text or speech and vice versa, enhancing accessibility for deaf students in Malawi. Features a real-time CNN-based recognition pipeline with 87% accuracy and a PyQt5 interactive UI for educational and broadcasting use cases.

Tools: Python, TensorFlow, CNN, OpenCV, PyQt5.
